# 🚀 TuranCodes AI Bootcamp

A modern and beautiful **AI Learning Bootcamp website** built with React and Vite.  
This project is designed to introduce students to Artificial Intelligence in a simple, practical, and beginner-friendly way.

---

## 🌟 About This Project

**TuranCodes AI Bootcamp** is a clean, responsive, and interactive website for promoting a 15-day AI learning workshop.

The website focuses on:

- Artificial Intelligence basics
- Beginner-friendly AI learning
- Practical activities
- Student-focused content
- Modern glass-style UI
- Smooth and attractive design

---

## 🎯 Main Features

- ✨ Modern macOS-style glass design
- 📱 Fully responsive layout
- 🎨 Beautiful dark theme UI
- 🧠 AI workshop sections
- 📅 Day-wise learning structure
- 🔗 Clickable cards and links
- ⚡ Fast performance using Vite
- 💻 Built with React.js

---

## 🛠️ Tech Stack

This project uses:

- React.js
- Vite
- JavaScript
- HTML5
- CSS3
- Tailwind CSS

---

## 📚 Bootcamp Topics

The bootcamp includes beginner-friendly topics such as:

- What is Artificial Intelligence?
- How AI works in daily life
- AI tools and responsible usage
- Prompt writing basics
- Creativity with AI
- Simple projects and activities
- Future career awareness
